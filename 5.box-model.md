# Box Model (modelo de cajas)

* Te darás cuenta de que (casi) todo es una caja CSS
* posiciones, tamaños, espaciado, bordes, colores
* Las cajas pueden estar una al lado de la otra, o una encima de la otra.
* Los elementos HTML son cajas
